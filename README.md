# Spotify-Dataset
Analyzed and visualized a Spotify dataset to explore trends in songs, artists, genres, and overall popularity. I cleaned and processed the data using Python (Pandas), handled missing values, and structured it for analysis. I created charts to show top genres, most-streamed artists, song trends over time, and popularity patterns, which helped me understand how different factors influence streaming performance. This project enhanced my data analysis, visualization, and insights extraction skills while working with real-world music data and practicing effective storytelling through charts and graphs.
